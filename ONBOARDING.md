# Onboarding

## Scope

This file provides a quick start for contributors and agents working in `guild`.

## First-session checklist

1. Verify repository state:
   - `git status --short --branch`
2. Read the local index:
   - `INDEX.md`
3. Confirm module/hive configuration:
   - `aas-module.json`
   - `aas-hive.json`
4. Review protocol baseline:
   - `protocols/AGENT_INTEROP_V1.md`
